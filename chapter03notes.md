# Chapter 3 - Introduction to C# App Programming

### Comments
#### Single Line
Use `//` then type comment
##### Example
```cs
// Comment here 
```
#### Multi-line
Use `/* */` to start and to end
##### Example
```cs
/* Comment here first line
   another line for commenting */
```
